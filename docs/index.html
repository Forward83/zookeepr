Welcome to the ZooKeepr documentation repository ================================================ Thanks for choosing to use or evaluate ZooKeepr as your conference management system. This documentation repository is designed to get you up to speed with; * Developing ZooKeepr - for Python, Postgres and other developers who wish to extend or leverage ZooKeepr functionality * Deploying ZooKeepr - for system administrators who wish to get a ZooKeepr instance up and running * Administering ZooKeepr - for conference administrators who which to use ZooKeepr to run their conference or other event
